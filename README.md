## Plugins

A continuación una imagen sobre los plugins agregados en la pagina web;

<img src=/Plugins.png/>

### EnvoExtra

Este plugin lo único que hace es agregar temas premium para que se puedan usar sin necesidad de ser premium.

### Polylang

La principal funcion de polylang es agregar un traductor a tu pagina web, es decir, dentro de la edición de tus paginas podrás agregar una traducción a esta misma, la cual dependiendo del idioma tendras que escribirlo tu a mano traducido para que luego aparezca a la hora de traducir dentro de la pagina web.

### WP Go Maps

Por último pero no menos importante, este plugin agrega un maps a tu web, puedes elegir el sitio del mundo que tu quieras y aparecera ahi una marca.
